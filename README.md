桥梁激光扫描仪基于三维激光扫描技术，利用激光测距原理，通过记录被测桥梁表面大量点的空间坐标等信息，为道路桥梁三维建模以及安全检测等其他应用提供真实数据。

元器件及模块：
STC8A8K64S4A12×1               MCU
TTL摄像头模块×2	                获取JPEG格式图片
激光测距雷达传感器模块×1	返回激光点距离信息
JDY-16蓝牙模块×2	                发送各类数据到接收端
GY-952六轴传感器×1	当前位置姿态解算
MAX485模块×1	                UART转RS485通信
DM4010直流无刷电机×3	带动扫描仪运作并返回确定位置编码
18650锂电池组×3	                12V直流供电

工作流程：
1.控制电机到达初始设定位置；
2.读取GY-952六轴传感器控制电机进行调平；
3.检查蓝牙连接情况，等待用户蓝牙设备连接；
4.接收用户端选项指令，比对指令模式识别；
5.进入相应模式执行；
6.执行完成后，等待下一指令。

各执行模式任务：
注：激光扫描仪底部电机负责横向（水平方向）旋转，带动激光雷达等旋转的电机负责纵向（竖直方向）旋转。扫描过程为 横向电机按指定步长转动到位记录位置编码，纵向电机按指定步长转动到位记录位置编码后向雷达请求点距离信息，横向电机保持位置不变，纵向电机重复这一过程直至完成该方向切片上相应角度（纵向）区域扫描，然后纵向电机回到初始位置，横向电机前进一步，重复上述过程在相应角度（横向）区域范围依次进行切片扫描。获得数据（一帧）格式为 横向位置编码，纵向位置编码，点距离。下述文字中“扫描”均为此意。
1.自动全部扫描模式。进入该模式，针对全范围（180°）进行扫描，并在完成扫描后进行指定间隔值的拍照回传。
2.拍照识别扫描模式。进入该模式，调用摄像头进行拍照回传，基于计算机视觉的方法检测图像中的扫描目标（桥梁）区域，根据摄像头参数推算目标点相对于扫描仪的方向角，针对该角度区域进行扫描，并在完成扫描后进行指定间隔值的拍照回传。
3.指定区域扫描模式。进入该模式，调用摄像头进行拍照回传并显示至用户端，由用户在图片中框选指定扫描区域，通过计算机视觉的方法检测图像中框选区域，根据摄像头参数推算目标点相对于扫描仪的方向角，针对该角度区域进行扫描，并在完成扫描后进行指定间隔值的拍照回传。
